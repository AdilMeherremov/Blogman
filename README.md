## About this project 

Blogman is a fully responsive blogging platform developed with next.js and supabase which users can create their own accounts and make, delete, edit their posts while being able to see each others posts

## Key features

- Authentification with supabase
- Post management
- User profiles
- CRUD operations
- Fully responsive design
- Clean and readable code
- Smooth and simple animations

## Tech Stack that I used for this project

- Framework: Next.js
- Database & Auth: Supabase
- Styling: Tailwindcss & Shadcn/ui
- Animation: Framer motion
- Language: Typescript
